# 📚 书籍转游戏 - 完整模板系统

## 🎯 这是什么？

这是一套**完整的、可直接执行的**模板系统，让任何人拿到一本书后，可以按照模板一步步给AI下指令，自动生成一个和 **growth-hacker-game 完全一样效果**的教学游戏。

**效果保证：**
- ✅ 相同的UI设计（暗色赛博朋克风格）
- ✅ 相同的游戏机制（时间压力、资源管理、多步决策）
- ✅ 相同的动画效果（数字滚动、进度条、弹窗）
- ✅ 相同的用户体验（新手引导、移动端适配、成就系统）
- ✅ 完全复制所有细节参数

### 核心文档 #2：DESIGN_SPECIFICATIONS.md

**所有参数的详细配置表。**

#### 何时使用？
- **需要修改某个参数时**：查表找到精确值
- **自定义主题时**：替换颜色/字体/间距
- **理解设计逻辑时**：看为什么这样设计

#### 如何使用？
```
1. 打开 DESIGN_SPECIFICATIONS.md
2. 找到你要修改的参数（如：主题色）
3. 复制参数值
4. 粘贴到你的CSS/JS文件中
5. 刷新浏览器查看效果
```

#### 示例
```
# 场景1：我想改主题色为红色
打开 DESIGN_SPECIFICATIONS.md
找到：--accent-primary: #A855F7
修改为：--accent-primary: #FF6B6B
保存 → 刷新 → 整个游戏变成红色主题

# 场景2：我想改按钮hover效果
打开 DESIGN_SPECIFICATIONS.md
找到：.action-btn:hover { transform: scale(0.98); }
复制到你的CSS文件
调整参数：transform: scale(0.95);（更明显的效果）
```

### 核心文档 #3：LEVEL_DESIGN_TEMPLATE.md

**关卡设计填空模板，最常用！**

#### 何时使用？
- **设计每个关卡时**：复制模板，填写内容
- **修改现有关卡时**：参考模板结构

#### 如何使用？
```
1. 复制一份 LEVEL_DESIGN_TEMPLATE.md
2. 重命名为 level-1-design.md
3. 填写【括号】中的所有内容
4. 完成后，整个粘贴给AI
5. AI生成JSON格式的关卡代码
6. 复制到 levels-data.js
```

#### 示例工作流
```
# 1. 填写基本信息
关卡编号：1
关卡主题：投资人危机
难度等级：简单
预计通关时间：3小时

# 2. 填写场景描述
时间：星期一，早上9:17
开场：你的手机响了三次...
（填写500-800字的场景）

# 3. 填写6个行动选项
关键行动 #1：
名称：深挖用户数据
描述：查看用户构成、流失明细
耗时：40分钟
价值：5星
反馈：【填写200-400字】

（填写剩余5个行动）

# 4. 填写3个问题
问题 #1：
提问者：Tom（投资人）
问题："我们能撑过这次危机吗？"
选项A（正确）：...
选项B（错误）：...
（填写完整）

# 5. 粘贴给AI
完整复制你填写的内容，发送：
"请基于以下关卡设计模板，生成完整的关卡代码（levels-data.js格式）："

# 6. AI生成代码
{
    id: 1,
    title: "⚠️ 关卡 #1：投资人危机",
    objective: "...",
    scene: { ... },
    actions: [ ... ],
    questions: [ ... ]
}

# 7. 复制到文件
粘贴到 levels-data.js 的 LEVELS_DATA 数组中
```

## 🎯 常见问题 FAQ

### Q1: 我完全不会编程，能用这套模板吗？
**A: 能！** 你只需要：
1. 复制Prompt → 粘贴给AI → 等待完成
2. 不需要理解代码，AI会全部完成
3. 你只需要提供书籍内容和填写关卡设计

---

### Q2: 需要多长时间完成？
**A: 看你的目标：**
- **MVP版本**（基础功能）：2-3天
- **完整版本**（所有功能）：5-7天
- **打磨版本**（高质量内容）：10-14天

**时间分配：**
- 30%代码（AI完成，很快）
- 70%内容（人工设计关卡，需要时间）

---

### Q3: 可以改成其他风格吗？（如：科技风、古典风）
**A: 可以！** 修改 DESIGN_SPECIFICATIONS.md 中的配置：

```
# 改成科技风（蓝色主题）
--accent-primary: #00D4FF → 蓝色
--accent-secondary: #7C3AED → 紫色
背景：加入电路板图案

# 改成古典风（棕色主题）
--accent-primary: #D4AF37 → 金色
--accent-secondary: #8B4513 → 棕色
字体：改成衬线字体
```

### Q4: 能用于商业项目吗？
**A: 可以！** 但要注意：
