# 📝 关卡设计填空模板

## 使用说明
1. 复制这个模板
2. 填写【括号】中的内容
3. 粘贴给AI，说："请基于这个模板生成完整的关卡代码"
4. 完成！

---

## 📋 关卡基本信息

```
关卡编号：【1-10】
关卡主题：【例如：投资人危机、产品崩溃、竞品攻击】
难度等级：【简单/中等/困难】
预计通关时间：【3小时/2小时/1小时】
```

## 🎯 任务目标设定

### 核心目标
```
玩家需要在【时间限制】内完成【核心任务】

例如：
- 3小时内准备好数据，说服投资人继续支持
- 2小时内找出产品崩溃原因，恢复服务
- 1小时内制定公关策略，平息舆论危机
```

### 失败后果
```
如果失败，将导致：
1. 【后果1】
2. 【后果2】
3. 【后果3】

例如：
1. 融资终止，公司可能关门
2. 核心团队离职
3. 用户信任崩塌
```

### 成功标准
```
通过条件：
- 完成至少【2-3】个关键调查
- 对话正确率≥【66%】
- 关键资源不耗尽（【现金/信任/声誉】）

评级标准：
- 5星：【100%正确 + 高效用时】
- 4星：【85%正确】
- 3星：【66%正确，勉强通过】
```

## 📖 场景描述（HOOK法则）

### 1. HOOK（钩子）- 吸引注意
```
时间：【星期几，上午/下午 具体时间】
地点：【会议室/办公室/咖啡厅】
开场：【一个吸引注意的细节或突发事件】

例如：
时间：星期一，早上9:17
地点：会议室
开场：你的手机响了三次。是CEO Sarah。"会议室，现在。Tom（投资人）也在。"
```

### 2. CONTEXT（情境）- 提供信息
```
数据面板（3-5个关键数字）：
1. 【数据1】：【具体数字】
2. 【数据2】：【具体数字】
3. 【数据3】：【具体数字】

例如：
1. 昨晚8点用户数：10,000人
2. 今早8点用户数：5,000人（-50%）
3. 当前流失速度：15%/天
4. 账户余额：$10,000
5. 月工资支出：$18,000
```

### 3. CHALLENGE（挑战）- 引入压力
```
关键人物（2-3个）：
1. 【角色】【姓名】：【他/她的态度/要求】
2. 【角色】【姓名】：【他/她的态度/要求】

对话（3-5句，制造紧迫感）：
【角色1】："【对话内容】"
【角色2】："【对话内容】"
【角色1】："【关键施压语句】"

例如：
Tom（投资人）："FitnessPro昨晚宣布永久免费了。"
Sarah（CEO）："我们的订阅是$20/月。"
Tom："我12点有电话会，到时候问你一个问题。你最好有数据支撑的答案。"
```

### 4. CHOICE（选择）- 倒计时
```
现在是【几点】，你有【X小时X分钟】准备。

选择你的行动...
```

## 💬 对话问题设计（2-3个）

### 问题 #1
```
提问者：【角色】【姓名】
问题：【关键问题，考察玩家是否掌握了核心数据】

选项A（正确）：
文本：【基于数据的正确答案】
反馈：【为什么正确，引用具体数据和逻辑】

选项B（错误）：
文本：【凭感觉的错误答案】
反馈：【为什么错误，缺乏什么数据】

选项C（错误）：
文本：【另一个错误答案】
反馈：【为什么错误】

选项D（错误）：
文本：【第三个错误答案】
反馈：【为什么错误】

例如：
提问者：Tom（投资人）
问题："竞品免费了，我们能撑过这次危机吗？"

选项A（正确）：
"能。我们的核心用户（30%）贡献80%收入，流失的主要是试用用户。"
反馈：
✅ 正确！

【数据支撑】
你通过用户数据分析发现：
- 核心用户（>30天）：1,500人，流失率仅6%
- 付费用户：900人，其中90%是核心用户
- 核心用户付费率：60%，月流失<5%

【逻辑推理】
竞品免费策略吸引的是价格敏感的低质量用户，这些用户本来就不是我们的目标群体。真正为产品价值付费的核心用户很稳定。

【关键洞察】
危机是表面的。只要守住核心用户，公司的长期价值不变。

Tom点头："很好，你有数据掌控力。我会继续支持你。"

选项B（错误）：
"能！我们的产品比竞品好，用户会回来的。"
反馈：
❌ 错误！

【问题所在】
这是凭感觉的判断，没有数据支撑。投资人不会被"相信我"这种话说服。

【数据反驳】
如果你做了用户数据分析，会发现：
- 流失的5,000用户中，94%是新用户
- 核心用户流失率很低（6%）
- 付费用户几乎没流失

【正确做法】
应该先分析用户构成和流失明细，用数据证明核心用户稳定。

Tom摇头："你没有掌控力。我需要重新考虑这笔投资。"

选项C（错误）：
"不好说，我需要更多时间观察。"
反馈：
❌ 错误！

这种模棱两可的回答暴露了你缺乏数据分析。投资人要的是明确的判断和数据支撑。

选项D（错误）：
"不能。我们需要立即降价跟进，否则所有用户都会流失。"
反馈：
❌ 错误！

【问题所在】
这是恐慌反应，不是基于数据的理性决策。

【数据反驳】
用户数据分析显示：
- 付费用户流失率很低（<10%）
- 核心用户留存率高（95%）
- 真正流失的是从未付费的试用用户

【正确做法】
不要被表面数据（总用户-50%）吓到，要深挖结构性数据。
```

## 🎯 通关评分标准

### 评分维度
```
1. 完成度
   - 完成【2-3】个关键行动：【及格】
   - 完成【所有】关键行动：【优秀】

2. 正确率
   - 【0-1】题正确：【失败】
   - 【2】题正确：【及格】
   - 【3】题正确：【完美】

3. 时间效率
   - 用时<【50%】：【速度奖励+20%】
   - 用时【50-80%】：【正常】
   - 用时>【80%】：【时间惩罚-20%】

4. 资源管理
   - 任何资源耗尽：【失败】

   - 资源充裕（>60%）：【奖励+10%】
```
