## 🔧 配置Claude API（可选）

### 为什么配置API？

- **不配置**：使用智能规则引擎（完全免费，体验良好）
- **配置后**：使用真实AI（更智能，每次对话都不同）

### 配置步骤

**1. 获取API Key**

访问：https://console.anthropic.com/
- 注册账号
- 创建API Key
- 复制 `sk-ant-...` 开头的密钥

**2. 配置文件**

复制模板：
```bash
cd /Users/mac/growth-hacker-game
cp api-config-template.js api-config.js
```

编辑 `api-config.js`：
```javascript
const API_CONFIG = {
    CLAUDE_API_KEY: 'sk-ant-你的密钥',  // 填入你的密钥
    USE_REAL_API: true,
    MODEL: 'claude-3-5-sonnet-20241022',
    // ...
};
```

**3. 刷新游戏页面**

重新打开 `game-mode-v3.html`，游戏会自动检测并使用真实API。

### 成本估算

使用Claude API的成本：
- 每局游戏约 30-50 次对话
- 每次约 500 tokens input + 300 tokens output
- **成本：约 $0.10-0.20 / 局**

**建议**：
- 先用免费的规则引擎模式体验
- 如果喜欢，再配置API享受更智能的AI

## 💡 游戏技巧

### 1. **平衡资源**
- 不要只关注增长，忽略团队精力
- 市场时机很重要，越早行动越好
- 用户信任影响转化率
- 品牌声誉影响病毒传播

### 2. **多轮对话**
- 不要急着做决定
- 多问AI顾问问题
- 听取不同NPC的意见
- 数据驱动决策

### 3. **策略组合**
- 短期+长期策略结合
- 获客+留存+变现并重
- 不要把鸡蛋放在一个篮子里

### 4. **学习成长**
- 每个决策后认真读反馈
- 学习增长黑客原则
- 应用到下一个决策

### 5. **关系管理**
- 保持各方满意度平衡
- CEO要速度，投资人要健康
- 用户要体验，CTO要质量
- 你需要平衡所有人

## 🐛 故障排除

### 问题1: 页面空白
**解决**：检查浏览器控制台（F12），看是否有JavaScript错误

### 问题2: 对话没有响应
**解决**：
1. 检查是否填写了任何内容再点发送
2. 查看游戏日志是否有错误提示
3. 刷新页面重试

### 问题3: API调用失败
**解决**：
1. 检查API Key是否正确
2. 检查网络连接
3. 游戏会自动降级到规则引擎模式

### 问题4: UI显示异常
**解决**：
1. 清除浏览器缓存
2. 使用Chrome/Safari/Firefox最新版本
3. 检查CSS文件是否加载
