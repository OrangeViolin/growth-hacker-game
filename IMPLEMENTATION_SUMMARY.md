### 3步集成
```html
<!-- 1. 添加脚本 -->
<script src="particle-system.js"></script>
<script src="combo-ui.js"></script>

<script>
// 2. 初始化

let comboUI = new ComboUI();

// 3. 使用
particleSystem.explodeFromButton(button);
comboUI.update(combo, true);
</script>
```
